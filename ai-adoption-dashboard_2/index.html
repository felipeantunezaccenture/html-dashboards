📐 Prompting Guide
Frameworks, attributes and best practices to raise your output quality immediately
The Framework
01
PURPOSE
Define the goal and what needs to be achieved
02
PRESENTATION
Indicate the expected output format and structure
03
PARAMETERS
Set your constraints and rules the model must follow
04
PERSPECTIVE
Provide context on yourself and your audience
05
POLISH
Refine and reiterate to enhance your result
Full Prompt Example
»
Identify the top 3 AI use cases for a retail client looking to reduce operational costs by 15% in the next 12 months.
»
Present as an executive summary table with columns: Use Case, Business Impact, Implementation Timeline, and Estimated ROI.
»
Focus only on proven AI technologies — no experimental solutions.
»
Assume you are a senior Accenture consultant presenting to the CFO and COO of a Fortune 500 retailer.
»
Review the output and prioritize use cases by highest ROI first.
General Considerations
💡
Clarity & Simplicity
Use clear, full sentences. Avoid unnecessary complexity.
🎯
Specificity
Be precise — generic prompts lead to generic responses.
Prompt Attributes
Prompt Length
e.g. 'more specific', '3 paragraphs'
Domain Info
e.g. age group, profession, interest
Avoid Acronyms
e.g. spell out 'S&OP' fully
Perspectives
e.g. 'objective', 'neutral', 'critical'
Tone of Prompt
e.g. 'casual', 'urgent', 'formal'
Audience
e.g. 'economic impact', 'productivity'
Define the Role — 'Flip the role'
e.g. 'devil's advocate', 'consulting partner', 'expert in supply chain'
The RCCO Framework
R
Role — set the persona and expertise level
C
Context — project, client, and goal
C
Constraints — length, tone, what to avoid
O
Output format — table, bullets, slide, code

A structured approach to consistently deliver executive-ready, high-quality outputs.

1
Structure Before Speed
Use Thinking mode and Projects to create consistency across outputs.
2
Define the Source of Truth
Clarify numbers vs narrative ownership before drafting.
3
Target Precisely
File name + exact location + task + format in every prompt.
4
Iterate with Intent
Refine like a manager, not like a user. Each prompt builds on the last.
5
Validate Before Exporting
Run a QA pass and assumptions check before sharing externally.
⚡ Power User Guide
ChatGPT advanced techniques used by the top-performing cohort on the team
⚙️
Use the Right Settings
Select the appropriate thinking mode and depth so the first pass has structure and logic. Choose memory type based on project needs.
📁
Maintain Project Discipline
One project per client or workstream, with clearly named source files so the model can reliably reference "what is true."
🎯
Be Explicit About the Outcome
Clearly state the result you expect and the level of quality required. Don't assume the model will infer your standards.
❓
Force Clarification Upfront
Ask the model to raise key questions before drafting to ensure accuracy. This saves one full revision cycle.
🎙️
Voice-to-Text + Precise Iteration
Brief thoroughly through spoken word, then edit outputs like a manager — iterate until client-ready.
🔍
Use Extended Thinking Wisely
Reserve it for hard multi-step strategy problems, complex quantitative tasks, or high-risk executive messaging.
Starter Prompt — Force Clarification
"Before you start drafting, ask me the 3–5 clarifying questions that would most improve the quality and accuracy of your output. Wait for my answers before proceeding."
Use this at the start of any complex task to eliminate rework.
